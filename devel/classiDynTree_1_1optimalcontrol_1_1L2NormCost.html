Evaluate cost second partial derivative wrt the state and control.

It is the result of \(\frac{\partial^2 g(t, x, u)}{\partial x \partial u}\), thus it has number of rows equals to the number of states and number of cols equal to the number of control inputs.

Parameters
[in]timeThe time at which the partial derivative is computed.
[in]stateThe state value at which the partial derivative is computed.
[in]controlThe control value at which the partial derivative is computed.
[out]partialDerivativeThe output partial derivative.
Returns
True if successfull, false otherwise (or if not implemented).
