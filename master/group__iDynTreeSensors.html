Predict the measurement of a set of sensors.

Given a SensorList object describing a list of sensor of a model fill the output argument predictedMeasurement with the predicted measurement of the sensors consistent with the state and the acceleration/torques and forces of the Model.

This function takes in input the internal buffers used for the computation to avoid dynamic memory allocation.

Parameters
[in]modelthe model used to predict the sensor measurements.
[in]sensorListthe sensors list used to predict the sensors measurements.
[in]traversalthe Traversal used for predict the sensor measurements.
[in]robotPosthe position of the model used for prediction.
[in]robotVelthe velocity of the model used for prediction.
[in]robotAccthe acceleration of the model used for prediction.
[in]gravitythe gravity acceleration (in world frame) used for prediction.
[in]externalWrenchesthe net external wrench acting on each link.
[out]buf_properRobotAccinternal buffer, storing the proper acceleration of the model.
[out]buf_linkPosinternal buffer, storing the position of every link in the model.
[out]buf_linkVelinternal buffer, storing the velocity of every link in the model.
[out]buf_linkProperAccinternal buffer, storing the proper acceleration of every link in the model.
[out]buf_internalWrenchesinternal buffer, storing the internal wrenches.
[out]buf_outputTorquesinternal buffer, storing the generalized joint torques.
[out]predictedMeasurementthe predicted measurements for the sensors.
Returns
true if the sensors in the list are all valid
