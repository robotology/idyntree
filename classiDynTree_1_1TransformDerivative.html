iDynTree::TransformDerivative class
#include <iDynTree/TransformDerivative.h>

Class representing the derivative of Transform object.

Note

Even if most methods documentation are written assuming that this class represents the derivative of a Transform with respect to time, it can be used to represent a derivative with respect to an arbitrary variable.
